컴퓨터와 운영체제 H/W (Device) <하드웨어> - 중앙처리장치 - 주기억장치 - 보조기억장치 - 입력장치 - 출력장치 S/W (Program) <소프트웨어> 프로그래밍 언어로 만들어진다. - 응용S/W -> 사용자의 목적에 맞게 작성된 대부분의 프로그램. - 시스템S/W -> 하드웨어를 제어하고 자원을 할당받아 응용프로그램이 동작하는 환경을 제공해주는 프로그램. 동작 환경 GUI(Graphic User Interface) - 그래픽 기반 CLI(Command Line Interface) - 명령줄 기반 Server - 서버용 프로그램이 설치가 되어 서비스를 제공하는 컴퓨터. Client - 클라이언트 프로그램으로 서비스를 사용하거나 요청하는 컴퓨터. - GUI환경도 제공하나 보통 CLI환경을 사용 - 가볍고 처리속도가 빠름 - 서버에 최적화된 운영체제
